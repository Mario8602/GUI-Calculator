# GUI-Calculator


***


Калькулятор Windows с графическим интерфейсом.


***

## Запуск приложения

1) Скачайте проект с Github
2) Активируйте виртуальное окружение

       python -m venv venv
    
3) Установите необходимые библиотеки с помощью

       pip install -r requirements.txt
     
4) Запустите файл 

       calculator1.py
